# BMW-Price-Analysis-PowerBI
Interactive Power BI dashboard analyzing BMW sales trends, models, fuel types, and regional performance.

## Project Overview
This project analyzes BMW car sales data using Power BI.  
The dashboard provides insights into car models, fuel types, regional sales, and yearly trends.

The goal of this project is to demonstrate data visualization and business insight generation using Power BI.

---

## Dataset
- Source: Kaggle
- Total Rows: 50,000
- Columns include:
  - Model
  - Fuel Type
  - Transmission
  - Region
  - Year
  - Price
  - Color

---

## Dashboard Features

### 1. Sales Overview
- Total number of cars sold
- Most popular model

### 2. Model Analysis
- Car count by model
- Average price by model

### 3. Fuel Type Analysis
- Petrol
- Diesel
- Hybrid
- Electric

### 4. Regional Analysis
Sales distribution across regions:
- Asia
- Europe
- North America
- Middle East
- Africa
- South America

### 5. Transmission Analysis
Comparison between:
- Automatic
- Manual

### 6. Yearly Sales Trend
Visualizes how car sales changed from 2010 to 2024.

---

## Tools Used
- Power BI
- Data Cleaning
- Data Modeling
- Data Visualization

---

## Key Insights
- The **7 Series** model has the highest average price.
- Sales are distributed evenly across regions.
- Automatic transmissions slightly dominate manual cars.
- Fuel types are almost evenly distributed.
